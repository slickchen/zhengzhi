📱 使用指南
第一步：选择复习课时
打开网页后，会看到8个课时的选择按钮

点击你想复习的课时（如"第一课 踏上强国之路"）

页面会自动加载该课时的所有默写题目

第二步：开始默写
填空题：

在空白处输入正确答案

注意答案的完整性和准确性

多选题空：

每个空都有对应的输入框

按顺序填写每个空的答案

第三步：检查与提交
完成所有题目后，可以：

点击"查看全部答案"检查自己的回答

或直接点击"提交答案并批改"

提交后系统会自动批改并显示成绩

第四步：查看结果
查看总题数、答对题数和得分率

查看每道题目的正确答案和解析

如有错误，仔细阅读解析加强记忆

📖 学习建议
日常复习
每日一课：每天安排15-20分钟复习一个课时

先默后查：先自己尝试默写，再看答案和解析

错题标记：重点关注默写错误的题目，加强记忆

考试前复习
全面复习：考前一周将所有课时都默写一遍

重点突破：针对错误率高的知识点进行重点复习

模拟测试：计时完成一个课时，模拟考试环境

学习技巧
理解记忆：不仅要记住答案，更要理解知识点

联系实际：将道法知识与现实生活联系起来

定期回顾：每周回顾一次之前默写错误的题目
